Capitulo 1: Qué es CSS

Introducción a CSS

CSS (Cascading Style Sheets) es un lenguaje basado en reglas que definen grupos de estilos.

Elementos de CSS: selector (de etiquetas HTML), declaración contiene propiedad (colores, fuentes, etc) y valor (para los colores, fuentes, etc); el conjunto de todo esto se llama regla.

Las reglas de CSS se usan para dar una estética agradable a los sitios web.
